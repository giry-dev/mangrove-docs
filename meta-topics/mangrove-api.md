# Mangrove API

{% hint style="info" %}
Wraps around [ethers.js](https://github.com/ethers-io/ethers.js). Works in the **web browser** and **Node.js**.
{% endhint %}

## Install

You can install the API using \`npm\` package manager using:

```bash
npm install @mangrovedao/mangrove.js
```
